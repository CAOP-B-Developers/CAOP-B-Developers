# Guia de Screenshots

## How to capture

1. Open the **CAOP-B** app on your phone
2. Navigate to the desired screen (with real data, avoid empty states)
3. Take a screenshot
4. Transfer to your computer
5. Place the files in `screenshots/` with the names below

## Required screenshots

| File | Screen | Tips |
|------|--------|------|
| `onboarding.png` | Onboarding first slide | Show the gradient and welcome text |
| `home.png` | Home Screen | Greeting, budget card, 4 categories, suggestions |
| `explore.png` | Category grid | Restaurant, Nightclub, Hotel, Resort cards |
| `details.png` | Establishment details | Photo, name, map, favorite button, reviews |
| `reservations.png` | Reservation screen | Date picker (14 days), time slot, party size |
| `map.png` | Map view | Multiple establishment pins on the map |
| `favorites.png` | Favorites screen | List of saved places with details |
| `agent-dashboard.png` | Agent dashboard | Metrics cards and today's reservations list |

## Optional screenshots

| File | Screen |
|------|--------|
| `menu.png` | Restaurant menu with items and prices |
| `checkout.png` | Cart/checkout with items |
| `auth.png` | Login screen |
| `notifications.png` | Notification center |
| `reviews.png` | Reviews screen |
| `profile.png` | User profile |

## Specifications

- **Format**: PNG
- **Aspect ratio**: 9:16 or full screen
- **Naming**: lowercase, no spaces or accents
- **Content**: Real Angolan establishment data recommended
